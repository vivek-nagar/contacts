## Features

- Add/Remove contacts
- Search in contacts
- contacts saved in local storage
- Cross platform

## Installation

flutter create your_app_name

```bash
  flutter create my-project
  cd my-project
```

This app is built with Flutter 2.2, with Sound Null Safety, and works out of the box on:

✅ Android
✅ iOS
✅ Web
